## Notes

- **Mode C**: each 6 h block reads `actual_weather[sample_hour = base_sample_hour + 6·block]` (Luo-2024 weather convention). Oracle / upper-bound planner.
- **SR save %** is `(SR − baseline)/baseline · 100`. Negative is savings.
- **Luo − SR mt** is the SOG-lock cost. Positive means the SR DP saves more.
- Default sog_step = 0.1 kn. Number of target SOGs = (v_max − v_min)/0.1 + 1.
